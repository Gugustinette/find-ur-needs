À Propos

Find Ur Needs est un site à but non lucratif dédié à l'entraide, entre les personnes pouvant donner et les personnes dans le besoin.

Il est possible de trouver tous les points proches de chez soi depuis l'accueil.
De plus, la demande d'ajout d'un nouveau point est possible via le formulaire (Google Form) disponible en cliquant sur le bouton "Ajouter un point".
